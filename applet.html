Applet

This applet can simulate a neural network with a variable number of hidden layers and neurons per layer. The network can also be trained using back propagation by setting the learning rate and momentum to convert polar coordinates to cartesain form and plot the total error as the network is trained.

Click here to launch the applet

Instructions

Setup Network

Here you can set parameters of the neural network: the number of layers and neurons in each hidden layer by using with the corresponding spin boxes. Click the Apply button to create a new neural network with the given parameters, or press Cancel to discard changes to the parameters and leave the current network unmodified.

When first started, the applet will use a network with 2 hidden layers of 15 neurons each.

Train Network

The current network can be trained by selecting the number of train-test iterations and the learning rate and momentum factor to use when training. Once the training is finished, a graph will appear showing the total percentage error along the y axis against the number of iterations on the x axis. This can be repeated to train the network any number of times.

Clicking the Reset Weights button will randomize the weights of the network, allowing you to train the same network again with different training parameters to see any difference in the results.

Test Network

Here you can see the actual output of the network by changing the values of r and theta. The network will run with the current values of r and theta as soon as either of the spinners are modified. The actual x and y outputs of the network are displayed along with the expected values.
